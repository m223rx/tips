                # Challenge: CSV Statistics Analyzer

## Difficulty
Medium

## Description
Create a Python program that reads a CSV (Comma‑Separated Values) file and generates a statistical summary for each numeric column. The summary should include the **count**, **mean**, **median**, **mode**, **standard deviation**, **minimum**, and **maximum** values. Your program should output the results in a clear, tabular format to the console.

The CSV file may contain headers, missing values, and a mix of numeric and non‑numeric columns. Your solution must gracefully handle these cases, ignoring non‑numeric columns and skipping missing entries when computing statistics.

## Requirements
- Accept the path to the CSV file as a command‑line argument.
- Detect and use the first row as column headers.
- Identify numeric columns automatically (e.g., columns that can be converted to `float` without errors).
- Skip missing or empty values when calculating statistics.
- Compute and display the following for each numeric column:
  - **Count** of non‑missing values
  - **Mean**
  - **Median**
  - **Mode** (if multiple modes exist, list all)
  - **Standard deviation**
  - **Minimum**
  - **Maximum**
- Present the results in a neatly formatted table (you may use libraries like `tabulate` or format manually).

## Bonus
- Add optional command‑line flags to:
  - Select specific statistics to display (e.g., `--mean`, `--median`).
  - Output the summary to a JSON or CSV file instead of printing to the console.
- Implement support for different delimiters (e.g., semicolon, tab) via a `--delimiter` argument.
- Provide an option to round the output values to a specified number of decimal places.
